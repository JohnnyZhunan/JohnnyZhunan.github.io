五子棋游戏大家都比较熟悉，因此，这里就不再对游戏规则进行过多的说明，只是说明开发该游戏用到的工具。并将代码和注释呈现给大家。

游戏所需资源

对于我，我就是在网上找了两个自己喜欢的图片。本人会一点蹩脚的PhotoShop，于是就将所找到的图片进行了一些处理。所挑选的图片可以按照个人喜好来选。此外，为了使游戏更加丰富，也可以选点好听的音乐。本人就下载了一首《高山流水》作为游戏的北京游戏，然后把这些资源导入到相应的位置，这些文件的所在位置需要注意，本人为了图省劲，所以全部放在了本地（工程文件下），如图所示。
